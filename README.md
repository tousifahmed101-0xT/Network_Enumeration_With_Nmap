## Overview

This repository documents network enumeration techniques using Nmap, written as a personal reference and study resource. The focus is not just on what commands to run, but on understanding what is actually happening at the network level: how packets are exchanged, what responses mean, and how to make sense of results in different environments.

The content is aimed at anyone working through penetration testing labs, preparing for certifications like OSCP or CEH, or just trying to build a solid foundation in network reconnaissance.

**Note:** Everything here is intended for use in authorized environments only. Do not scan networks or systems without explicit permission.

## Requirements

- Linux system (Kali, Parrot, or Ubuntu work well)
- Nmap 7.x or later ([nmap.org](https://nmap.org))
- Root or sudo access for most scan types
- A lab environment or authorized target. [HackTheBox](https://hackthebox.com) and [TryHackMe](https://tryhackme.com) are good options

---

## How to use this

Go through the topics in order. Each section assumes you have read the one before it. The goal is to build understanding progressively rather than just collect commands to copy-paste.
